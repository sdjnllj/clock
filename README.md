# 时钟学习应用

一个帮助孩子学习认识时间的交互式网页应用。

## 功能特点

- 📚 学习模式
  - 手动设置时间
  - 隐藏数字时间，通过"显示答案"按钮检查
  - 预设时间按钮快速练习

- ⏰ 实时模式
  - 显示当前真实时间
  - 时钟实时走动
  - 数字时间显示

- 🎨 界面设计
  - 清晰的时钟刻度
  - 明暗主题切换
  - 平滑动画效果
  - 响应式布局

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript

## 使用方法

1. 打开 index.html 即可使用
2. 默认为学习模式，可以：
   - 通过输入框设置时间
   - 使用预设时间按钮
   - 点击"显示答案"查看正确时间
3. 点击"切换到实时模式"可以查看当前时间
